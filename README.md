# Payment-Web-App-PHP-Stripe
Web Application for accepting payments with Stripe and storing details on Database. 
<br>This application utilises:
<ul>
<li>PHP</li>
<li>PHP PDO</li>
<li>JavaScript</li>
<li>Stripe API</li>
<li>MYSQL</li>
<li>Bootstrap 4.0 </li>
</ul>
